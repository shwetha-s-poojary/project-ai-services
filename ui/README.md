# ui

UI for AI Services

## Pre-requisites

Before you start, make sure you have the following installed:

- [Node.js](https://nodejs.org/) **v18 or later**
- [Yarn](https://yarnpkg.com/) **v1.22 or later+**

Verify versions:
```bash
node -v
yarn -v
```

## Getting Started

1. Clone the repository:

   ```bash
   git clone <repository-url>
   ```

2. Navigate to the project directory:

   ```bash
   cd ui
   ```

3. Install dependencies:

   ```bash
   yarn install
   ```

4. Run the development server:

   ```bash
   yarn run dev
   ```

5. Open the application in your browser at `http://localhost:5173`.

## Scripts

- `yarn run dev`: Starts the development server.
- `yarn run build`: Builds the application for production.
- `yarn run preview`: Previews the production build.
- `yarn run lint`: Runs ESLint to check for code quality issues.
